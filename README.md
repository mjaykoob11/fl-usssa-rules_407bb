# FL USSSA Baseball Rules Assistant
### 10U · 11U · 12U · 13U Divisions

A self-contained rules reference tool for Florida USSSA Baseball. Covers the most common rules questions for coaches, managers, and parents — no app install, no login, no API key required.

---

## How to Use

**Option 1 — View directly in GitHub (read-only preview):**
Open `index.html` in the GitHub file browser. GitHub renders basic HTML inline.

**Option 2 — Run locally (full experience, recommended):**
1. Click the green **Code** button → **Download ZIP**
2. Unzip the folder
3. Open `index.html` in any web browser — that's it

**Option 3 — GitHub Pages (shareable live link):**
1. Go to repo **Settings → Pages**
2. Set Source to `main` branch, `/ (root)`
3. Click Save — your link will be: `https://YOUR-USERNAME.github.io/fl-usssa-rules_407bb/`
